Projeto App de Receitas! :poultry_leg:
Aplicação mobile para busca de alimentos e bebidas.

- Utilizado React, Redux e CSS.

## Instruções

Primeiramente faça o clone do projeto e depois:

```bash
- npm install
```

```bash
- npm start
```
